# AdvanSol Optimizer

Home Assistant custom integration for AdvanSol optimizers through a TCP RS485 bridge.

After installation, restart Home Assistant and add **AdvanSol Optimizer** from **Settings -> Devices & services**.

Use this repository as a HACS custom repository with category **Integration**.

If HACS reports **401 Unauthorized**, re-authenticate HACS with GitHub or configure a valid GitHub token in HACS. The repository itself is public.
